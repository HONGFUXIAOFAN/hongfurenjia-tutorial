# 红福人家系列产品使用教程

## 快速开始

### 方法一：直接打开（离线可用）
双击 `index.html` 文件，即可在浏览器中打开，无需网络连接。

### 方法二：启动本地服务器预览
```bash
# 进入目录
cd 红福人家使用教程_网页版

# 使用 Python 启动（macOS/Linux）
python3 -m http.server 8080

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8080
```
然后访问 http://localhost:8080

## 分享给他人的方法

### 1️⃣ 压缩后直接发送（最简单）
将整个文件夹压缩成 `.zip` 文件，通过微信/QQ/邮件发送，对方解压后双击 `index.html` 即可使用。

### 2️⃣ 上传到免费托管平台（推荐长期分享）

#### GitHub Pages（永久免费）
1. 注册 GitHub 账号
2. 创建新仓库 `hongfurenjia-tutorial`
3. 上传 `index.html` 到仓库的 `main` 分支
4. 设置 → Pages → Source: main branch
5. 等待 2-3 分钟，获取地址如：`https://你的用户名.github.io/hongfurenjia-tutorial/`

#### 飞书文档嵌入
1. 打开飞书文档
2. 插入 → 网页
3. 粘贴 GitHub Pages 或其他托管地址

#### 微信小程序（需企业资质）
联系技术支持获取小程序嵌入方案
